Detailed Description

Klasa reprezentujaca drzewo poszukiwan binarnych (BST).

Klasa BST umozliwia wykonywanie operacji na drzewie, takich jak dodawanie, usuwanie, wyszukiwanie sciezki do wezla oraz wyswietlanie drzewa w roznych kolejnosciach.
